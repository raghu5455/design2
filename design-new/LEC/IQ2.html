iQ2 Overview

iQ2™ is LEC’s complete, end-to-end IIoT platform solution, designed from an industrial automation perspective to address individual needs and industry challenges. In collaboration with leading network carriers and top-tier hardware providers, we bring our cutting-edge IIoT platform to virtually any location demanding the capture and rapid response to mission-critical data.

iQ2 Basics

  • Use of non-proprietary/off the shelf edge hardware
  • User Interface offers exceptional usability for less experienced staff
  • A mobile-first approach allows for a fully functional solution on any mobile device.
  • Easily integrates with existing PLC’s and other control systems
  • Integration with other process software such as traditional SCADA and GIS
  • Is quickly and easily implemented and deployed
  • Requires no programming but rather drop-down driven configuration

Value of iQ2

iQ2 is the most complete and comprehensive automation system. With iQ2, you have access to:

  • Highly affordable remote monitor and control solutions
  • Advanced notice of pending process and equipment failures
  • Intuitive user interface for all levels of users
  • Data Collection from all major PLC’s and HMI’s
  • Visualize and analyze data in near real time
  • Reduce unnecessary truck rolls with an HMI in your pocket
  • Integrates data collection with traditional SCADA systems and GIS

iQ2 Features

iQ2’s user acceptance helps drive value with its contemporary mobile interface, flexible and adaptable backend architecture, customizable dashboard graphics and widgets, with on-demand and automatic data file transfers. iQ2 includes unlimited field asset mapping, multiple device site support, active/passive GPS, and tailored built dashboards and pages. iQ2 offers detailed user event logs and audit trails tracking all configuration changes and integrates with other solutions.
